# AI Product Workflow Memo

Static bilingual GitHub Pages site for Yang Chenghui's AI product workflow application materials.

## Local Preview

Open `index.html` directly, or run:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## GitHub Pages Deployment

1. Create a new GitHub repository, for example `followup-candidate-memo`.
2. Upload all files in this folder to the repository root.
3. In GitHub, open `Settings -> Pages`.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save and wait for the GitHub Pages URL.

## Files

- `index.html`: page content
- `styles.css`: page styles
- `assets/`: resume PDF and anonymized project screenshots
